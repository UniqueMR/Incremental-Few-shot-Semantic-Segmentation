# Incremental Few-shot Semantic Segmentation

![Overall](https://github.com/UniqueMR/Incremental-Few-shot-Semantic-Segmentation/blob/main/img/overall.png)

This project is an implementation of incremental few-shot semantic segmentation based on [CaNet](https://arxiv.org/abs/1903.02351) and [EHNet](https://arxiv.org/abs/2207.12964).
